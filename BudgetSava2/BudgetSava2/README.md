# 💰 MoneyWays

💰 Budget Tracker App (BudgetSava)
📌 Project Overview

BudgetSava is a mobile Android application developed in Kotlin that helps users manage their personal finances efficiently. The app allows users to track expenses, categorize spending, and monitor their financial goals using a clean and user-friendly interface.

This project focuses on functionality, data persistence using Room Database, and usability.

🎯 Features
🔐 User Authentication
Register with name, email, date of birth, and password
Login using email and password

📂 Category Management
Create and use predefined categories (e.g., Groceries, Transport, Utilities)
Each category is assigned a unique color for easy identification

💸 Expense Management
Add new expenses with:
1.Amount
2.Category
Description
Date
Start and End Time
Attach an optional photo to each expense
📷 Photo Support
Users can upload an image for each expense
Images can be viewed when browsing expense entries
📅 View Expenses by Period
Users can filter and view expenses within a selected date range
Helps track spending over specific periods
📊 Category-Based Reporting
Displays total amount spent per category
Helps users understand spending patterns
💰 Budget Goals
Users can set:
Minimum monthly spending goal
Maximum monthly spending limit
💾 Data Persistence
All data is stored locally using Room Database (SQLite)
Ensures offline access and data reliability
🛠️ Technologies Used
Kotlin – Programming language
Android Studio – Development environment
Room Database – Local data storage
XML – UI design
MPAndroidChart – Data visualization (graphs)
📱 Application Flow
Register a new user
Login to the app
Add categories (or use default ones)
Add expenses with optional images
View expenses by selected date range
Analyze total spending per category
Monitor monthly budget goals
