# 💰 BudgetSava

BudgetSava is a simple and user-friendly Android application developed in Kotlin that helps users manage their personal finances. The app allows users to track expenses, categorize spending, and monitor their monthly budget goals efficiently.

## 🚀 Features

### 🔐 User Authentication
- User registration with name, email, date of birth, and password  
- Secure login using email and password  

---

### 📂 Categories
- Predefined expense categories (e.g., Groceries, Entertainment, Transport)  
- Each category has a unique color for easy identification  

---

### 💸 Expense Management
- Add expenses with:
  - Amount  
  - Category  
  - Description  
  - Date  
  - Start & End Time  

---

### 📷 Photo Attachment
- Option to attach a photo to each expense  
- Stored locally and accessible when viewing expenses  

---

### 📅 Filter by Date
- View expenses within a selected date range  
- Helps track spending over specific periods  

---

### 📊 Reports & Insights
- View total amount spent per category  
- Analyze spending patterns easily  

---

### 💰 Budget Goals
- Set minimum monthly spending goal  
- Set maximum monthly spending limit  

---

### 💾 Offline Data Storage
- Uses Room Database (SQLite)  
- All data is stored locally on the device  

---

## 🛠️ Tech Stack
- Kotlin
- Android Studio
- Room Database (SQLite)
- XML Layouts
- MPAndroidChart

## ⚙️ Setup Instructions

1. Clone the repository:
https://github.com/henryowusu01/BudgetSava.git

2. Open the project in Android Studio 

3. Sync Gradle and build the project

4. Run the app on an emulator or physical device

Register → Login → Dashboard → Add Expense → View Reports
